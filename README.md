# 📚 Book Library App

![Node.js](https://img.shields.io/badge/Node.js-Express-green)


![Status](https://img.shields.io/badge/status-active-success)

A simple **Node.js + Express** application to manage your personal book collection.
Keep track of what you’ve read, what you’re reading, and what’s still waiting on your shelf.

---

## ✨ Features

* ➕ **Add books** to your library
* ✅ **Mark books as Read / Unread**
* 🗑️ **Delete books** anytime
* 🔎 **Filter books by status** (Read / Unread)
* 📖 **Search books by title or author**

## 🚀 Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/book-library-app.git
cd book-library-app
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start the server

```bash
node server.js
```

### 4️⃣ Open in your browser

```
http://localhost:3000
```
